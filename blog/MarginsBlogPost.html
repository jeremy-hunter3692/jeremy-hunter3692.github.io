CSS Padding, Margins and Borders:

Padding, Margins and borders are methods of creating space around an element with some key differences. Lets start with Margins and padding first. At it's simplest; Margins are a way of creating emtpy space around an element and Padding is a way of creating space inside your element.

It is a lot easier to understand the difference between these ideas when we think about about the element as a block shape rather than, say, just some text. Take a box shape. Margins will change the space outside of the box, padding will change the space inside of the box. You can picture it here as this box:

If you look at the red line (this is actually the border, which we will come back to but for now we will just use it to locate and speak about the edge.) Margins go from the red line out and Padding goes from the red line in.

Now lets think about it specifcally as some text in a box. In fact specfically all this text in these boxes. We have some white text with a background colour. The distance/space between this text and the start of the background-colour box is the padding. The distance between the edge of that box and the bigger box surrounding it is the margin.

To be more accurate both Margins, padding and borders add space to your element. This means that padding will change the size and of your element/block (and therefore sometimes the shape) where as Margins will change the distances between that block/element and another block/element.
